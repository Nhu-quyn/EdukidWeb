# Edukid English Web

Edukid is an interactive English learning platform for kids. It combines games, videos, and quizzes to make learning more engaging and enjoyable. A leaderboard system is included to encourage continuous learning.

---

## Key Features

- Interactive topic-based learning
- Mini games & quizzes for practice
- Video-based lessons
- Student leaderboard
- Google authentication
- JWT-based authentication & authorization

---

## Tech Stack

Frontend: React, Redux  
Backend: Node.js, Express  
Database: MongoDB  

---

## Architecture

MVC Pattern + Service Layer

Client → API → Service → Database

---

## Setup

Backend:
npm install
npm start

Frontend:
cd frontend
npm install
